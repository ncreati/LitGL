# litGL
A Python package to draw text in an OpenGL frame.


## Install litGL ##
Run command:
    
    pip3 install litGL 

For the bleeding edge install with:

    pip3 install git+https://github.com/ncreati/litGL

Run example code:

    python3 -m litGL.examples.example1
    ...
    python3 -m litGL.examples.example5
